# Features

## 1. User Financial Profiling
- Collects user data such as income, expenses, savings, and risk preference
- Stores data for future interactions

## 2. Conversational Interface
- Chat-based interaction
- Step-by-step questioning to gather information

## 3. Intent Detection (NLP)
- Identifies user intent such as saving, investment, or budgeting
- Uses keyword-based or simple NLP techniques

## 4. Decision-Making System
- Applies rule-based logic to analyze user financial condition
- Generates appropriate financial suggestions

## 5. Personalized Recommendations
- Advice is based on user-specific data
- Not generic responses

## 6. Response Generation
- Uses templates or NLP models to generate human-like responses

## 7. Data Storage
- Stores user profile in database (MongoDB)

## 8. Safety & Disclaimer
- Includes disclaimer for financial advice
- Avoids risky or misleading suggestions
