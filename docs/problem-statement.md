# Problem Statement

In today’s digital world, many individuals struggle with managing their personal finances effectively. While there is a large amount of financial information available online, most users lack personalized guidance based on their own income, expenses, and financial behavior.

Existing chatbot systems provide generic responses and fail to consider the user's financial condition before suggesting actions such as saving or investing. This can lead to impractical or even misleading advice.

Additionally, many users are not comfortable using complex financial tools or consulting professional advisors due to cost or lack of awareness.

Therefore, there is a need for an intelligent system that:
- Understands user queries using Natural Language Processing (NLP)
- Collects relevant financial information
- Provides personalized and practical financial suggestions

This project aims to address these challenges by developing a Smart Financial Assistant Chatbot that combines NLP with rule-based decision-making to deliver context-aware financial guidance.
