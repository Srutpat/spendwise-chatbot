# System Flow

## Step 1: User Interaction
- User sends a message through chat interface

## Step 2: Input Processing
- Message is sent to backend
- NLP module identifies intent

## Step 3: Check User Profile
- System checks if user financial data is available
- If not, chatbot asks relevant questions

## Step 4: Data Collection
- Collects:
  - Income
  - Expenses
  - Savings
  - Risk preference

## Step 5: Decision Making
- Applies predefined rules to analyze financial condition

## Step 6: Response Generation
- Generates a personalized response

## Step 7: Output
- Response is sent back to frontend and displayed

## Flow Summary

1. User sends message  
2. System detects intent  
3. System checks/collects data  
4. Logic is applied  
5. Response is generated  
6. User receives answer
