# Objectives

The main objectives of this project are:

- To design and develop a chatbot that can understand user queries using NLP techniques
- To collect and maintain a structured financial profile of the user
- To analyze user financial data such as income, expenses, and savings
- To provide personalized financial suggestions based on predefined rules
- To implement a conversational interface that interacts naturally with users
- To ensure the system gives safe and general financial guidance, not professional advice

## Scope of the Project

The chatbot will:
- Provide basic budgeting advice
- Suggest saving strategies
- Recommend simple investment options (like SIP)

The chatbot will NOT:
- Provide professional financial or legal advice
- Predict stock market trends
- Perform real-time financial transactions
