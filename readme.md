# Smart Financial Assistant Chatbot

This project is an NLP-based chatbot designed to provide personalized financial guidance based on user input such as income, expenses, and financial behavior.

## Tech Stack
- Frontend: React
- Backend: Node.js, Express
- Database: MongoDB
- NLP: Rule-based + HuggingFace

## Features
- Financial profiling
- Budget analysis
- Investment suggestions
- Conversational interface

## Note
This chatbot provides general financial guidance and should not be considered professional financial advice.
